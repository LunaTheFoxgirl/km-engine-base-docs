Font.drawVertical

engine render texture font Font

Basic string draw function for right-to-left vertical text

If font does not have vertical metrics the implementation will try to estimate some rounded to the nearest pixel

This only works for languages where text is monspaced (Japanese and Chinese for example) As well only languages that are read right-to-left when vertical.
